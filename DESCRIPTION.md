# Repository Description

Short description (GitHub About field):

Real-time boid flocking simulation in C++ with OpenGL, ImGui controls, and optional OpenMP/MPI parallel execution.

Long description:

A C++17/OpenGL implementation of Craig Reynolds' boids model that visualizes emergent flocking behavior using alignment, cohesion, separation, and obstacle avoidance. The project includes an interactive ImGui settings panel and supports serial, OpenMP, and MPI execution modes for performance comparison and experimentation.
